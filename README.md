
# PDF RAG Chatbot

This is my first GenAI project.

I built a chatbot that answers questions from uploaded PDF files using RAG (Retrieval-Augmented Generation).

Tech Stack:
- Python
- Streamlit
- PyPDF
- Sentence Transformers
- ChromaDB
- Groq API

Live Demo:
